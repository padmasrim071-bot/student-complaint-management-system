# Student Complaint Management System

## Project Overview

The Student Complaint Management System is a web-based application developed using Java Spring Boot and PostgreSQL.

This system allows students to:

* Register and Login
* Submit Complaints
* View Complaint Status
* Search Complaints
* Manage Student Profile

The Admin can:

* View All Complaints
* Update Complaint Status
* Delete Complaints
* View Statistics Dashboard
* Monitor Complaint Records

## Technologies Used

* Java
* Spring Boot
* PostgreSQL
* HTML
* CSS
* JavaScript
* IntelliJ IDEA
